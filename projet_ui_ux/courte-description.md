# Courte description

- Outils utilisés : HTML, CSS, React, Vite et des composants UI modernes pour construire l’interface.
- Choix techniques : architecture en étapes, gestion d’état avec React Context, et persistance locale des données du formulaire.
- L’interface a été pensée pour être claire, responsive et adaptée à un parcours de réservation simple.
- Une logique bilingue français/anglais a été intégrée pour améliorer l’accessibilité du projet.
- Difficultés rencontrées : la gestion des accents et des séquences Unicode a demandé des corrections sur plusieurs écrans.
- Le projet est prêt pour un déploiement sur Vercel avec une configuration légère et standard.
