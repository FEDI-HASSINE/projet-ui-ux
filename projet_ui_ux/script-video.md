# Script vidéo de présentation

Bonjour, aujourd’hui je vais vous présenter notre projet de réservation de contrôle technique pour l’ATTT.

Ce projet est une interface web moderne développée avec HTML, CSS, React et Vite, avec des composants réutilisables pour garder un code clair et organisé.
L’application a été pensée pour offrir une expérience simple, rapide et responsive, afin de guider l’utilisateur étape par étape jusqu’à la confirmation du rendez-vous.
Nous avons également intégré une logique bilingue français/anglais pour rendre l’interface plus accessible.
Côté technique, les données du formulaire sont conservées localement pour permettre de reprendre la réservation à tout moment.

Je commence par la page d’accueil, qui affiche un design propre avec le logo ATTT et un bouton pour démarrer ou reprendre la réservation.
Ensuite, l’utilisateur choisit le type de service, puis le centre souhaité parmi plusieurs options avec des indicateurs de disponibilité.
La troisième étape permet de sélectionner la date et l’heure du rendez-vous grâce à un calendrier et à des créneaux horaires.
Après cela, l’utilisateur remplit ses informations personnelles, comme le nom, le CIN, le téléphone et les informations du véhicule.

Une fois les données saisies, la page récapitulative permet de vérifier toutes les informations avant validation.
L’utilisateur peut aussi revenir directement à une étape précédente pour modifier un choix si nécessaire.
Enfin, la page de confirmation affiche un numéro de réservation, les détails du rendez-vous et des actions utiles comme le téléchargement du récapitulatif ou l’ajout au calendrier.

Pendant le développement, nous avons rencontré quelques difficultés, notamment sur la gestion des accents français et l’affichage bilingue.
Ces problèmes ont été corrigés pour garantir une interface plus propre et plus professionnelle.
Le projet est maintenant prêt pour un déploiement sur Vercel.

Merci pour votre attention.
