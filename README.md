# projet-ui-ux

Projet de réservation ATTT développé avec React, Vite et une interface bilingue FR/EN.

Le code source principal se trouve dans le dossier [projet_ui_ux](projet_ui_ux), qui contient l'application complète, les composants, les écrans et la configuration de déploiement.

Pour lancer le projet en local :

```bash
cd projet_ui_ux
npm install
npm run dev
```

Pour générer la version de production :

```bash
cd projet_ui_ux
npm run build
```
